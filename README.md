Kanban Board - Task Management Web Application

This project demonstrates the use of drag and drop concept in JS.
It helps organize task in To do, In Progress, and Completed.
Users can:

- Add new tasks
- Arrange Tasks according to To do, In Progress, Completed.
- Delete particular tasks

Technology Stack Used:

- HTML
- CSS
- Bootstrap
- JavaScript

Extra feature:

- Show counts according to tasks classification(to do, in progress, completed).
- Save tasks in browsers localStorage
- Provide date and time when new task is created.

JavaScript concepts revised:

- Drag and Drop
- DOM Manipulation
- Event handling/Listeners
- Form validation
- localStorage
- Array iteration
